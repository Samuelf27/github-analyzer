## 📌 Sobre o projeto

Digite um nome de usuário e o **GitHub Analyzer** consome a **API pública do GitHub** para montar um dashboard com o perfil, métricas e visualizações de dados — útil para recrutadores, devs e curiosos avaliarem um perfil rapidamente.

## ✨ Funcionalidades

- 👤 Cartão de perfil (avatar, bio, localização)
- 📈 Métricas: repositórios, estrelas totais, seguidores e seguindo
- 🍩 **Gráfico de linguagens** mais usadas (doughnut)
- 📊 **Gráfico de estrelas** por repositório (barras)
- ⭐ Lista dos repositórios mais populares (estrelas, forks, linguagem)
- 🎨 Cores de linguagem fiéis às do GitHub

## 🛠️ Tecnologias

JavaScript (Fetch API, async/await) · [Chart.js](https://www.chartjs.org/) · API pública do GitHub

## 🚀 Como rodar

```bash
git clone https://github.com/Samuelf27/github-analyzer.git
cd github-analyzer
# abra index.html no navegador (ou use Live Server)
```

> 💡 A API pública do GitHub permite ~60 requisições/hora sem autenticação — suficiente para uso casual.
